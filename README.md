# 目線｜日文假名學習網站

## 專案介紹

目線是一個以繁體中文使用者為主要對象的日文學習網站，專注於假名辨識與對照練習。
目前提供平假名、片假名與混合模式的互動式卡片練習，讓學習者能夠在不同分類下進行觀察與回想。

本專案目前聚焦在以下方向：
- 假名辨識
- 平假名與片假名對照
- 遮擋練習
- 分類練習
- 日後擴充生活日文單字
